# Mehedi Amin - Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Quick Start

### Option 1: VS Code (Recommended)
1. Press `Ctrl+Shift+P` → type "Tasks: Run Task" → press Enter
2. Select "Start Live Server"
3. Open http://localhost:3000 in your browser

### Option 2: Terminal
```bash
npm start
```

### Option 3: Direct Open
Simply double-click `index.html` to open in your browser.

## 📁 Project Structure

```
My-portfolio/
├── index.html      # Main HTML file
├── style.css       # External styles
├── script.js       # JavaScript functionality

```

## ✨ Features

- **Typing Animation** — Dynamic text cycling through roles
- **Responsive Design** — Works on mobile, tablet, desktop
- **Code Display** — Syntax-highlighted project cards
- **Smooth Animations** — Fade-in effects on scroll
- **Dark Theme** — Modern dark color scheme
